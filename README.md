# Prime Integer Relations

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy to Vercel

This is a Vite + React SPA. The repo includes a Vercel rewrite configuration so React Router routes work on refresh.

- Build Command: `npm run build`
- Output Directory: `dist`
